```shell
#yarn
yarn create expo-app

#npx
npx create-expo-app@latest

#pnpm 
pnpm create expo-app

#参数
--template

default
默认模板。旨在构建跨屏应用。包括推荐工具，如 Expo CLI、Expo Router 库和启用的 TypeScript 配置。适用于大多数应用。

blank
无需配置导航即可安装所需的最低限度的 npm 依赖项。

blank-typescript
启用了 TypeScript 的空白模板。

tabs
安装并配置基于文件的路由，并启用 Expo Router 和 TypeScript。

bare-minimum
生成带有本机目录（ android和ios）的空白模板。npx expo prebuild在设置期间运行。
```
